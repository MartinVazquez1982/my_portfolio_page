Computer Vision in Medicine: Development of a Baseline Segmentation Model for Brain Aneurysm Treatment Devices from 3D Medical Images

We developed a baseline segmentation model using U-Net to identify brain aneurysm treatment devices in 3D medical images. The project involved deep learning implementation with PyTorch, data preprocessing, training optimization, and integration with a web service. Our goal was to automate and improve accuracy in a traditionally manual process.
